# 老虎機數學入門 PART VI

來源：https://ezslotdesign.com/slotmath6/

PART VI 的主題是 **獨立滾輪老虎機**。

這類遊戲的特徵是：

```text
每一格都是一個獨立滾輪。
```

它和一般 5 軸輪帶老虎機最大的差異是：

```text
一般老虎機：
同一軸的 3 格畫面來自同一條輪帶的連續 stop。

獨立滾輪：
每一格都有自己的輪帶，所以每一格可以獨立抽取。
```

因此它比較像台灣水果盤的機制：任意盤面組合都有可能出現。

---

## 1. 獨立滾輪的基本概念

以 3 x 3、5 線遊戲為例，盤面可以標成：

```text
1 2 3
4 5 6
7 8 9
```

每一個格子都可以有自己的輪帶。

例如：

```text
格 1 有自己的輪帶
格 2 有自己的輪帶
格 3 有自己的輪帶
...
格 9 有自己的輪帶
```

所以這種遊戲不是「一軸拉出三格」，而是「九格各自抽一格」。

---

## 2. RTP 計算仍然從一條線開始

雖然架構不同，但計算邏輯仍然和一般 Line Game 類似。

先選一條線，例如上排：

```text
1 - 2 - 3
```

然後把這三格視為一條 3-reel line game。

計算方式：

```text
1. 統計每一格輪帶上的符號個數。
2. 把 Wild 加到可替代的符號中。
3. 計算各圖案 3 連線的出現次數。
4. 出現次數 / 所有組合數 = 出現機率。
5. 出現機率 × 賠率 = 該圖案 RTP contribution。
```

公式可以寫成：

```text
某圖案出現次數
= 第1格可中該圖案的 stop 數
 × 第2格可中該圖案的 stop 數
 × 第3格可中該圖案的 stop 數
```

```text
出現機率
= 出現次數 / 所有組合數
```

```text
期望值
= 出現機率 × 賠率
```

## 3. 為什麼同一欄通常使用相同輪帶

文章中特別提到：

```text
獨立滾輪老虎機通常會將同一欄的滾輪安排成相同的輪帶。
```

原因是為了讓每一條線的期望值相同，簡化計算。

例如 3 x 3、5 線：

```text
Line 1：1-2-3
Line 2：4-5-6
Line 3：7-8-9
Line 4：1-5-9
Line 5：3-5-7
```

如果同一欄使用相同輪帶：

```text
格 1 / 4 / 7 使用同一組輪帶
格 2 / 5 / 8 使用同一組輪帶
格 3 / 6 / 9 使用同一組輪帶
```

那麼每一條線都會經過：

```text
左欄輪帶
中欄輪帶
右欄輪帶
```

因此每條線的 RTP 都相同。

這時只需要算一條線：

```text
Total RTP = 單線 RTP
```

如果是用總押注來看：

```text
總期望得分 = 單線期望得分 × 線數
總押注 = Bet per line × 線數
Total RTP = 總期望得分 / 總押注
```

因為每條線期望值相同，所以最後仍等於單線 RTP。

## 4. 96.85% RTP 範例

文章中的 96.85% 範例是：

```text
大小：3 x 3
線數：5 線
Bet：1
同一欄使用相同輪帶
```

計算流程如下。

### 1. 單線組合數

每格輪帶長度都是 20。

一條線有 3 格，因此：

```text
所有組合數 = 20 × 20 × 20 = 8000
```

### 2. 加入 Wild 替代後的符號數

例如：

```text
WA = W + A
WK = W + K
WQ = W + Q
WJ = W + J
W10 = W + 10
```

某圖案的 3 連線出現次數：

```text
WA1 × WA2 × WA3
WK1 × WK2 × WK3
...
```

### 3. 算各圖案期望值

例如 A 的 3 連線：

```text
出現次數 = WA1 × WA2 × WA3
出現機率 = 出現次數 / 8000
期望值 = 出現機率 × A 的賠率
```

其他圖案同理。

最後把所有圖案的期望值加總：

```text
單線 RTP = 96.85%
```

因為 5 條線期望值相同：

```text
Total RTP = 96.85%
```

## 5. 同欄不同輪帶時，要逐條線計算

如果同一欄中存在不同輪帶，例如第 5 格輪帶被改掉：

```text
格 1 / 4 / 7 不再完全等價
格 2 / 5 / 8 不再完全等價
格 3 / 6 / 9 不再完全等價
```

那麼每條線經過的輪帶組合不同，期望值也會不同。

文章中的例子是：

```text
Line RTP:
96.850%
128.925%
96.850%
128.925%
128.925%
```

這時正確總 RTP 不是只看其中一條線，而是五條線平均：

```text
Total RTP
= (96.850% + 128.925% + 96.850% + 128.925% + 128.925%) / 5
= 116.095%
```

所以獨立滾輪遊戲如果同欄輪帶不同，就必須：

```text
逐條 line 計算 RTP，
再取所有 line 的平均。
```

## 6. 適合搭配的遊戲特色

獨立滾輪最大的特色是：

```text
任何盤面組合都有可能發生。
```

一般輪帶老虎機因為每軸畫面來自連續 stop，所以某些盤面不可能出現。

但獨立滾輪每格獨立抽取，因此比較適合搭配：

```text
全盤獎
任意位置收集
特定格位觸發
中間區域觸發
```

例如全盤獎需要畫面 9 格都可能自由組合，這種特色通常就很適合獨立滾輪架構。

---

## 7. 免費遊戲觸發要特別小心

獨立滾輪遊戲中的 Free Game 觸發方式可能和一般遊戲不同。

例如：

```text
盤面中間出現 3 個 Scatter 才觸發。
```

這種規則不是一般的：

```text
全畫面 Scatter 總顆數
```

而是指定某些格位。

所以計算前要先確認：

```text
Scatter 是看全畫面？
還是只看特定格位？
是否只看中間線？
是否每格都有不同輪帶？
```

若不同線或不同格位使用不同輪帶，仍然要依照實際格位逐一計算機率。

## 總結

PART VI 的核心可以整理成：

```text
獨立滾輪 = 每一格都有自己的輪帶。
```

計算邏輯：

```text
仍然從單條 line 出發，
用每格輪帶的符號數計算中獎組合數與期望值。
```

同欄相同輪帶時：

```text
每條 line 的 RTP 相同，
只要算一條線即可。
```

同欄不同輪帶時：

```text
每條 line 的 RTP 可能不同，
必須逐條計算後取平均。
```

設計意義：

```text
獨立滾輪讓任意盤面都有可能發生，
因此特別適合全盤獎、指定格位觸發、任意位置收集等特色。
```
