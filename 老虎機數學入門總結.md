## PART II

FG Retrigger 特色的 RTP 計算。

```text
Total RTP
= Base Game RTP + Free Game RTP
```

```text
Free Game RTP
= 觸發 FG 的機率
  × 觸發後平均總 FG spins
  × 單次 FG spin RTP
```

若 3S / 4S / 5S 給的初始 spins 不同：

```text
Free Game RTP
= (
    P(3S) × E[FG spins | 3S trigger]
  + P(4S) × E[FG spins | 4S trigger]
  + P(5S) × E[FG spins | 5S trigger]
  )
  × FG spin RTP
```
